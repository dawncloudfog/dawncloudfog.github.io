第四课 分支结构

我们的js代码都是顺序执行的

逻辑分支就是根据我们设定好的条件，来决定要不要执行某些代码

if 条件分支结构

通过if后面的条件进行判断，是否执行后面的代码

if (true){

document.write(“我是true所以执行”)

}

if~else 语句

if（false）{

document.write(“因为是false所以不执行”)

}else{

document.write(“因为前面是false所以我执行”)

}

break和continue循环符号

break终止循环

再循环没有进行完毕的时候，因为我设置的条件满足，提前终止循环

比如：我准备吃五个包子，在吃到第三个的时候不能再吃了，我就不吃了

要终止循环，就可以直接使用break关键字

continue结束本次循环

在循环中结束本次循环，继续执行剩下的循环

比如我准备吃五个包子，第三个掉地上了，不吃了，继续吃第四个第五个包子

跳过本次循环，可以使用continue关键字
