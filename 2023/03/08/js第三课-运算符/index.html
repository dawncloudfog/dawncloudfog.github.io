js第三课 运算符

发表于2023-03-08|更新于2023-03-08|javascript
|阅读量:

第三课 运算符

就是在代码里面进行运算的时候使用的符号，不只有数学运算，js里面有很多的运算方式

数学运算符

只有符号两边都是数值的时候才会进行加法运算，

var a = 10

var b = true

console.log(a+b) //结果为11

赋值运算符

给变量赋值，并且变量结果为最后一次赋值的值

变种

+= -= *= /= %=

number = number+1 number+=1

number = number -1 number-=1

number = number *2 number*=2

number = number /2 number /=2

比较运算符

> < <= >=

//结果为布尔类型的值（true or false)

赋值运算符的变种

==

比较符号两边的值是否相等，不管数据类型

===

比较符号两边的值和数据类型是否相等

！=

比较符号两边的数据是否不等于，不考虑数据类型

！==

比较符号两边的数据是否不等于，考虑数据类型

逻辑运算符

返回值为布尔类型

1、&&（与）

左右都为真返回为真

其中有一假则为假

var category = “衣服“

var = price = 300

console.log(category===”家电“ && price >200)

2、||（或）

符号两边只要有一个true则为true

var day = 5

console.log(day===7||day===5)

3、！(取反)

将数值布尔值取反输出

特殊

1、!!a ==>转换成布尔值，一个为取反，两个就变成了原值的布尔值

2、&& || 的短路用法

var y //这里忘记赋值，为undefined类型，不可以转换类型(全局项目视角)

console.log(y.toString()) //强制转换为字符串会报错，导致后面代码不执行

console.log(y&&y.toString()) //若y没有赋值，则会传出y的数据类型，后面也不会影响，若为真则可以执行

如果一个网站要注册会填个人介绍，如果没写则会输出“这个人很懒，什么也没有留下”

var z = “”

console.log(z||”这个家伙很懒，什么也没有留下”)

自增自减运算符

++

自增运算

分为两种：前置自加和后置自加

前置++，会先把值进行加一再将值返回

后置++，会先把值返回再进行自加

单独使用++num或者num++没有区别

但是如果在其他运算函数里面使用就会有输出先后的问题

–

自减运算

和自加运算类似

三元运算符

三元运算就是用两个符号组成一个语句

语法： 条件？条件为真的时候执行 ：条件为假的时候执行

例如：

var age = 18;

age>=18? alert(“已经成年”) ：alert(“没有成年”)
