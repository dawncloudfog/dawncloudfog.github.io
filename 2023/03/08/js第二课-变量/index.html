js第二课 变量

发表于2023-03-08|更新于2023-03-08|javascript
|阅读量:

第二课 变量

定义符号：var

几种定义方法：

var x = 200

var y，z

var x = 1, y = 2, z = 3

变量命名规则与规范

规则：必须遵守，不遵守就是错

1、变量名称可以由数字、字母、英文下划线、美元符号组成

2、严格区分大小写

3、不能由数字开头

4、不能是保留字或者关键字

5、不要出现空格

规范：建议遵守（开发者默认），不遵守不会报错

1、名称尽量有意义（语义化）

2、遵循驼峰命名规则，由多个单词组成的时候，从第二个单词进行首字母大写

3、不要使用中文

数据类型（重点）

  • 是指我们存储在内存中的数据的类型
  • 我们通常分为两大类 基本数据类型 和 复杂数据类型

基本数据类型

1、数值类型（number）

  • 一切数字都是数字类型
  • NaN是一个非数字

2、字符串类型（String）

  • 被引号包裹的所有内容（包括单引号）

3、布尔类型（boolean）

  • 只有两个 true 或者 false

4、null类型（null）

  • 复杂类型内的对象类型没有想好可以给它赋值为null,相当于空值

5、Undefined类型

  • 变量声明了但是没有赋值

数据类型的检测

使用console.log(typeof a)或者console.log(typeof (a))

例如：

var a = 1000;

console.log(typeof a) //结果是用字符串形式显示

​ //console.log(typeof typeof a)这种结果一定是字符串类型

console.log(typeof a +100) //结果是number+100

console.log(typeof（a+100)) //若a为数字则结果计算后输出为number

若变量未定义则输出为undefined ，若变量为null则输出为object

数值类型的转换

1、number（变量）

例如

var a =”100”

var b = Number(a)

console.log(a,b)

如果转化不成数值则输出NaN(not a number)

Boolean类型变化为数值 true为1 false为0

null类型为0

undefined为NaN

2、parseInt(解析整数)

此外，还有parseInt方法比Number方法判断标准宽松，它可以一位一位去判断是否为数值并且进行转换

例如

var a =”123abc”

var b =parseInt (a)

console.log(b) //这种结果为123

这种方法也有弊端，这种方法只能解析整数，小数情况只会转换前面的整数

3、parseFloat(解析浮点数)

这种方法可以转化小数形式的数值

例如

var a =123.45abc

var b =parseFloat(a)

console.log(b) //这种方法不仅可以转化小数，整数也可以完美转换

拓展：非加号转化数值类型

其他类型加上0则会输出类型内容加上数值内容

例如

var a = “杨志飞”

var b = a + 0

console.log(b) //会输出“杨志飞0”为字符串类型

若把+换成其他运算符则会将数值格式的其他类型转化为数值类型

例如

var a = “100”

var b = a-1

console.log( b) //这种会输出数值类型的99，相当于进行了类型转换，若不想改变值则-0 /1 *1即可

总结 ：各种方法各有优劣，按需取用即可

字符串的转换

1、String方法

和Number使用方法相同 //所有类型都可以转换

例如

var a =100

var b =String(a)

console.log(b)

2、toString 方法

和String方法使用类似,但是与String不同，undefined和null转化不了，运行会出问题，注意选用

例如

var a =100

var b = toString(b)

console.log(b)

3、加号方法

使用加号拼接字符串即可使前面的非字符串值转化为字符串

例如

var a = true

var b =a+””

console.log(b)

布尔类型的转换

在js中除了空字符串（不能有空格）、0、null、undefined、NaN这些是false以外其他都是true

代码见

var a = 100

var b = Boolean(a)

console.log(b)
